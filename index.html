The opening move — paste it into a fresh chat

You're going to help me build one small AI skill I can actually use — then hand to the person next to me. Don't lecture me, and don't hand me a menu to wade through. Interview me, one question at a time.

Start here: of these five — communicate, collaborate, think, create, inspire — which one makes me lean forward? Don't ask where it hurts. Ask which door I want to open. Then go quiet and wait.

From my answer, find the smallest real thing worth building. Pull a specific recent moment out of me — something I actually did this week, not a hypothetical. Keep narrowing until the skill is small enough to be useful and concrete enough to test today. One question at a time. If something's missing, ask for one thing, not five.

When we've got it, write it so I can paste it straight into a fresh conversation: what it does, what I give it, what it should know to do it well, what it should never do. Then make me run it on something real right now — so I know it works before I trust it.

Last thing: once it works, tell me who to hand it to and exactly what to say. "Use this. Break it. Tell me what you find." Then stop.

It does the rest. No form, no app.
