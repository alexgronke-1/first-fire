01 Center a draft

Ask the room what they keep redoing by hand. Open a shared chat, paste this in, and let the AI run the interview. It builds the draft. You carve it.

You're going to help a group build one real AI skill together — something they'll carry back to their teams. Don't lecture us. Interview the room, one question at a time.

Start with one question: what does this group keep redoing by hand? Something that happens often enough to have a name, and small enough to build a working tool for today. Then go quiet and wait.

From the answer, find the smallest real thing worth building. Pull a specific recent example out of the room — something someone actually did this week, not a hypothetical. Keep narrowing until the skill is small enough to be useful and concrete enough to test in the next twenty minutes. One question at a time. If something's missing, ask for one thing, not five.

When we've got it, write it so anyone in the room can paste it into a fresh conversation: what it does, what you give it, what it should know to do it well, what it should never do. Write it rough — the room is going to carve it.

Then stop.

Rough on purpose. Polished drafts invite politeness.
02 Carve it

Work through three questions in order. One at a time, out loud. Give quiet voices the first word on each.

What's wrong with it?
What does it get wrong, overfit, or assume? Name the thing, don't workshop it.
What's missing?
What would you need before you'd actually trust it to do this?
What should it never do?
Name the guardrail. The one thing that would make this go badly.
03 Run it

Pick one person's actual work — something in their inbox right now. Run the carved skill on it. See what it does.

If it holds, you're done. If it breaks, carve it again.

04 Carry it

Before you leave: each person names one person they'll show it to, or one channel they'll drop it in. Say it out loud.

"Use this. Break it. Tell me what you find."

Not because anyone was told to. Because something useful passed between people.
